### 交易模式提取（量化规则）

**模式1：大跌建仓**
- 机器人(562500)：从¥0.99跌到¥0.90-0.91区间（-9%），开始建仓（2025-03-23买入400股@0.907）
- 科创智能(588790)：从¥0.95+跌到¥0.80附近（-15%+），2025-11-06建仓2300股@0.798
- 规律：ETF跌10-15%以上时，视为"大跌"，开始建仓

**模式2：分批建仓**
- 机器人：分5次以上买入（200→400→200→100→200→300→300...），不是一次性
- 科创智能：建仓2300股后，再买入1600股@0.777（更低价格）
- 规律：跌得越深，买得越多；先试探性小仓位，确认趋势后再加仓

**模式3：高抛低吸/波段**
- 机器人(562500)：在0.9-1.0区间大量买入，在1.0-1.1+区间卖出
- 互联中概(159605)：最活跃，1.16-1.22区间反复买卖
- 规律：上涨过程中分批卖出，锁定利润；不等到最高点才卖

**模式4：反弹加仓**
- 机器人：2025-04-07在0.909买入200股（大跌后的反弹），然后4-20、4-23继续买入
- 规律："观望走势"——不急于全仓，确认反弹趋势后逐步加仓

**模式5：情绪高点减仓**
- 机器人：2025-05-07卖200@1.059，2025-05-12卖200@1.134，2025-05-18卖100@1.16
- 在持续上涨后，分批卖出，不是一次清仓
- 互联中概：2025-10-27清仓（可能是判断情绪到顶）

**量化规则总结：**
```
IF 大盘跌 > 3% AND ETF跌 > 10%:
    试探性建仓 (10%仓位)
    
IF ETF继续跌 (每跌5%):
    加仓 (20%仓位)
    
IF 反弹确认 (3天连涨 OR 突破20日均线):
    加仓至满仓
    
IF 上涨 > 20% (目标收益):
    减仓30% (锁定利润)
    
IF 情绪指标 > 0.8 (宝妈都入场):
    继续减仓50% (逃顶)
    
IF ETF属于情绪泡沫类:
    减仓比例翻倍 (泡沫风险高)
```

## 三、执行阶段

### Phase 1: 重构基准（等权买入持有）
- 修改主脚本：基准从沪深300改为24只ETF等权
- 买入持有策略：第一天等权买入所有ETF，之后不动
- 成本：只算一次买入成本

### Phase 2: 增强热点赛道选股
- 增加60日/120日动量
- 增加板块相对强弱（vs 沪深300）
- 增加板块内部排名（同类ETF中排前N）
- 权重重新分配

### Phase 3: 融入用户持仓策略
- 实现"大跌建仓"：沪指-3% + ETF-10% → 试探建仓
- 实现"分批建仓"：试探10%→加仓20%→满仓
- 实现"上升潮加仓"：3天连涨/突破均线 → 加满
- 实现"目标收益减仓"：涨20% → 减仓30%
- 实现"情绪高点减仓"：情绪>0.8 → 减仓50%

### Phase 4: 黄金对冲优化
- 恐慌时：黄金30%（用A股波动率代理VIX）
- 大跌时：黄金20%
- 牛市时：黄金0%

### Phase 5: 三因子分析程序（市场代理版）
- 情绪因子：波动率、成交量、动量代理
- 政治因子：避险资产走势、汇率波动代理
- 政策因子：市场宽度、趋势稳定性代理

### Phase 6: 情绪泡沫 vs 业绩支撑识别
- 情绪ETF分类：AI/机器人/芯片/云计算（高波动）
- 业绩ETF分类：消费/医药/金融/银行（低波动）
- 泡沫识别：短期涨幅>50% + 换手率>3倍均值
- 策略差异：泡沫ETF在情绪高点多减，业绩ETF少减

### Phase 7: 整合回测 + 优化迭代
- 整合所有模块到主脚本
- 回测验证
- 参数网格搜索优化
- 评价Agent评估

### Phase 8: 可选 — 评价Agent + Kimi subagent
- 评价Agent输出详细报告
- Kimi subagent分析策略弱点
- 提供改进建议
