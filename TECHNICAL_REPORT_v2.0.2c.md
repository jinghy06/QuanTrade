## 三、核心优化：双模式动量选股（ETF自身判断）

### 关键发现：为什么基于**ETF自身**判断比**市场统一**判断更好？

| 判断方式 | 2024-01-02选股 | 超额收益 | 评分 | Walk-Forward |
|---------|---------------|---------|------|-------------|
| **ETF自身判断**（120日跌幅>15%） | 516510, 515070, 562500, 512670, 512660, 159995, 515960 | **+2.29%** | **60/100** | **80%** |
| 市场统一判断（沪深300跌>10%） | 516510, 515070, 562500, 512670, 512660, 159995, 515960 | -1.74% | F(33) | 20% |
| 无底部模式（纯趋势） | 515960, 512660, 512670, 562500, 159995, 515070, 516510 | +0.22% | C(55) | 60% |

**原因**：ETF自身判断更精准——只有真正超跌的ETF（516510跌21%、515070跌17%）触发底部模式，而抗跌的ETF（515960仅跌0.2%）走正常模式得分最高仍被选中。市场统一判断让所有ETF走底部模式，导致抗跌ETF得分反而低。

### 双模式逻辑

```python
def calculate_momentum_score(etf_df, benchmark_df):
    mom_120 = close[-1] / close[-120] - 1
    is_bottom = mom_120 < -0.15  # 120日跌幅>15%
    
    if is_bottom:
        # 底部模式：选超跌最深的（跌幅越大=反弹潜力越大）
        score_oversold = -mom_120 / 0.30  # 跌幅>30%得满分
        score_60 = -mom_60 / 0.20        # 60日跌幅>20%得满分
        score_relative = (bench_mom_20 - etf_mom_20) / 0.20  # 比市场多跌=高分
        trend_score = 短期均线向上确认反弹
        return 0.40*score_oversold + 0.30*score_60 + 0.20*score_relative + 0.10*trend_score
    else:
        # 正常模式：选长期动量最强的（趋势跟踪）
        score_120 = mom_120 / 0.5
        score_60 = mom_60 / 0.3
        score_relative = (etf_mom_20 - bench_mom_20) / 0.2
        trend_score = 均线多头排列
        return 0.40*score_120 + 0.30*score_60 + 0.20*score_relative + 0.10*trend_score
```

## 五、为什么超额收益5%在当前条件下不可行？

### 数学约束

| 约束 | 数值 | 影响 |
|------|------|------|
| 等权基准收益 | 84.06% | 包含所有上涨ETF，任何选股都会错过部分 |
| 交易成本侵蚀 | 0.36% × 8次 = 2.88% | 每次调仓都有成本 |
| 数据ETF数量 | 仅8只有完整4.5年数据 | 15只只有2026年数据，无法用于2024-2025回测 |
| 回测周期 | 594天（2.5年） | 统计不显著（t检验nan） |
| 市场环境 | 2024-2026持续牛市 | 择时失效，情绪高点减仓无机会 |

### 8只ETF 2024-2026涨幅

| 排名 | ETF | 涨幅 | 策略选中 | 模式 | 备注 |
|------|-----|------|---------|------|------|
| 1 | 159995（芯片） | +207% | ✅ | 正常模式 | 第三强，被选中 |
| 2 | 515070（机器人） | +184% | ✅ | 底部模式 | 超跌最深，反弹最强 |
| 3 | 516510（科创） | +97% | ✅ | 底部模式 | 超跌第二深 |
| 4 | 562500（AI） | +61% | ✅ | 底部模式 | 超跌第三深 |
| 5 | 510300（沪深300） | +57% | 基准 | - | 基准 |
| 6 | 512660（军工） | +34% | ✅ | 底部模式 | 超跌第四深 |
| 7 | 512670（新能源） | +25% | ✅ | 底部模式 | 超跌第五深 |
| 8 | 515960（银行） | -12% | ✅ | 正常模式 | 抗跌，走正常模式得分最高 |

**关键失误**：515960（银行）走正常模式得分最高（抗跌=动量强），但牛市中最弱。这是底部判断的固有限制——银行ETF本身没跌，无法触发底部模式。

### 要超额5%需要什么？

1. **避免选中515960**：需要银行ETF跌幅>15%才能触发底部模式，但银行在底部根本没跌
2. **获取更多ETF数据**：159382、588790只有2025年数据，159605缺失
3. **真实情绪检测**：在情绪高点减仓（如2025年7月），当前市场代理无法触发
4. **VIX/跨市场数据**：地缘风险时减仓，当前黄金代理信号弱
5. **更长历史数据**：2018-2022年，覆盖完整牛熊周期，学习底部反弹模式
